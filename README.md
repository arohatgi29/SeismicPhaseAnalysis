# SeismicPhaseAnalysis

This repository contains reproducible code and examples for research on seismic phase analysis using circular statistics.

The main goal of this repository is to support reproducible workflows for analyzing seismic phase behavior, measuring phase coherence, and developing phase-based quality-control and masking methods for seismic data.
